# DogBazar — Торговая платформа для продажи и обмена собак

Торговая платформа, где пользователи могут покупать, продавать и обменивать собак. Реализована система объявлений, предложений обмена (офферов), контрпредложений, баланса пользователей и ролевая модель (USER/ADMIN).


## Технологии

- Java 17, Spring Boot, Spring MVC, Spring Data JPA, Spring Security
- PostgreSQL, Docker
- JUnit 5, Mockito, MockMvc, Maven

## Функциональность

**Публичная часть**
- Просмотр собак с фильтрацией (порода, редкость, цена)
- Просмотр активных объявлений
- Регистрация и вход

**Личный кабинет (USER)**
- Мои собаки, создание объявлений
- Предложения обмена: создать, принять, отклонить, контрпредложение
- Пополнение баланса, редактирование профиля

**Админ-панель (ADMIN)**
- Полный CRUD для собак
- Просмотр всех пользователей и объявлений
